# LCR 元件测量系统项目概览

## 1. 功能需求

### 1.1 核心功能（必须实现）

1. 接入未知电阻、电容或电感后，系统能够自动判断元件类型并显示主要参数值。
2. 输出完整复阻抗结果：
   - 阻抗幅值 `|Z|`；
   - 相位 `phase`；
   - 串联等效电阻 `Rs`；
   - 电抗 `X`；
   - 电感 `L` 或电容 `C`。
3. 使用现有 AD9910 产生正弦激励，正常测量时直流偏置为 0。
4. Blue 同步采集电压和电流信息，不能再用单路电压幅值直接猜测元件值。
5. 支持自动选择激励频率、参考电阻量程和激励幅值。
6. 支持 OPEN、SHORT、LOAD 校准并保存校准编号。
7. 在标定量程内达到题目要求的 1% 测量精度。
8. 测量失败、过量程、低信噪比或未校准时必须明确显示状态，不能把无效结果显示为 0。

### 1.2 次要功能（后续实现）

1. 手动选择 R、L、C 或 AUTO 模式。
2. 手动指定频率、激励幅值、平均次数和参考电阻档位。
3. 显示 ESR、Q 值、损耗角正切 D、并联等效参数。
4. 支持两点或多点扫频，检查元件参数随频率的变化。
5. 保存最近测量结果和校准时间。

### 1.3 用户操作

1. 用户按 `B` 进入 LCR 页面。
2. 接入被测元件后按 `D` 开始自动测量。
3. 屏幕依次显示量程选择、稳定、采样、计算和最终结果。
4. 主结果以大字显示，例如 `C = 9.98 nF`，下方显示测试频率、ESR、相位、量程和有效状态。

## 2. 非功能需求

### 2.1 精度

- 题目目标：L、C、R 测量误差不超过 1%。
- 建议首轮保证范围：
  - R：10 Ω～100 kΩ；
  - C：1 nF～100 µF；
  - L：100 µH～1 H。
- 扩展范围在完成夹具和多频点标定后再确定，不在未经验证时承诺 1%。

建议误差预算：

| 来源 | 目标上限 |
| --- | ---: |
| 参考电阻与温漂 | 0.15% |
| 两路模拟前端复增益残差 | 0.30% |
| ADC 噪声和重复性 | 0.20% |
| 夹具寄生与算法残差 | 0.30% |
| 总体设计目标 | ≤0.70%，给1%验收留余量 |

### 2.2 速度

- 单次4096点采样在2.5 MSPS下约1.6384 ms。
- 固定量程单点测量目标小于200 ms。
- AUTO自动量程和双频确认目标小于1 s。

### 2.3 可用性

- 测量过程中普通信号源配置不得覆盖当前激励。
- 任意异常必须恢复到安全激励幅值并释放测量状态机。
- 屏幕显示实际应用频率、实际量程和校准状态。

### 2.4 可维护性与扩展性

- 采集、复数拟合、阻抗计算、元件判定、标定和界面分别建模块。
- 所有物理量在协议中使用整数和明确单位，禁止跨节点传输浮点数。
- 保留原始两路复数相量用于调试和标定。

## 3. 约束与假设

1. Black 继续负责键盘、业务状态机和 AD9910 激励设置。
2. Blue 负责量程开关、ADC同步采样、阻抗计算和屏幕显示。
3. FPGA不参与基础LCR计算；后续可用于同步触发或高速采样扩展。
4. 当前Blue只有 `PC0/ADC1_IN10` 一路测量输入，完整方案需要增加第二路同步ADC输入。
5. 第二路候选为Blue的空闲ADC引脚，例如 `PC1/ADC2_IN11`；实施前必须核对开发板是否引出、LCD扩展板是否占用及电气连接。
6. 当前PC0还被对数检波模块使用。正式LCR测量时必须将对数检波与精密交流采样分开，或通过可标定的模拟开关选择，不能直接共用同一模拟节点。
7. 仅使用一个ADC并通过模拟开关顺序采两路可作为调试降级方案，但不作为1%精度主方案。

## 4. 方案设计

### 4.1 系统结构

```text
Black键盘/业务状态机
        │
        ├── 控制AD9910：频率、幅值、相位、启停
        │
        └── UART V2测量命令
                    │
                    ▼
             Blue测量控制器
        ┌───────────┼────────────┐
        │           │            │
     量程继电器   双ADC同步采样   屏幕显示
        │           │
        ▼           ▼
AD9910 ── 被测Zx ── Rref ── GND
  │          │          │
  └─ Vin采样 ┘          └─ Vr采样
```

推荐测量拓扑：

```text
Vin ── Zx ── Vr节点 ── Rref ── GND
```

- ADC通道A测量 `Vin`。
- ADC通道B测量参考电阻上的电压 `Vr`。
- 两路必须由同一个定时器边沿同步采样。

### 4.2 模拟前端

1. 两路采用尽量一致的衰减、偏置、运放缓冲和抗混叠网络。
2. 双极性交流信号偏置到ADC中点，建议约1.65 V。
3. ADC输入必须限制在0～3.3 V，并设置限流和钳位保护。
4. 测量激励从200 mVpp起步，可在50 mVpp～1 Vpp内自动调整；不得默认把7 Vpp大信号直接送入测量夹具。
5. 根据被测阻抗限制测试电流，建议首版不超过10 mA。
6. 参考电阻建议使用0.1%、低温漂器件，初始档位：

```text
10 Ω、100 Ω、1 kΩ、10 kΩ、100 kΩ
```

7. 量程切换优先使用低寄生继电器。若使用模拟开关，其导通电阻和寄生参数必须包含在每档复数标定中。

### 4.3 ADC采集

1. 使用STM32H750的ADC1和ADC2双规则同步模式。
2. TIM6 TRGO同时触发两路ADC。
3. DMA保存4096组同步样本，建议使用32位打包方式保存ADC1/ADC2数据。
4. 默认采样率2.5 MSPS；测量频率高于约500 kHz时需要单独验证ADC带宽、采样保持时间和前端相位误差。
5. 每个量程至少平均4帧；快速预判可以只采1帧。

### 4.4 复数相量计算

对每一路样本按实际激励频率和实际采样率拟合：

```text
y[n] = a*sin(2*pi*f*n/Fs) + b*cos(2*pi*f*n/Fs) + c
```

采用余弦参考定义复数相量：

```text
Y = b - j*a
```

分别得到：

```text
Vin = Bvin - j*Avin
Vr  = Bvr  - j*Avr
```

经过两路复增益标定后计算：

```text
I  = Vr / Rref
Vx = Vin - Vr
Zx = Vx / I
   = Rref * (Vin / Vr - 1)
```

随后得到：

```text
R = real(Zx)
X = imag(Zx)
|Z| = sqrt(R*R + X*X)
phase = atan2(X, R)
```

元件参数采用串联等效模型：

```text
X > 0：L = X / (2*pi*f)
X < 0：C = -1 / (2*pi*f*X)
```

同时保留：

```text
电感Q = X / Rs
电容D = Rs / |X|
电容Q = 1 / D
```

### 4.5 元件类型判断

1. 若 `|X|` 小于 `max(0.1*|R|, noise_floor)`，判定为电阻。
2. 若X显著为正，判定为电感。
3. 若X显著为负，判定为电容。
4. AUTO模式至少使用两个频点确认：
   - 电阻的 `|Z|` 应基本不随频率变化；
   - 电容的 `|X|` 应近似与频率成反比；
   - 电感的 `|X|` 应近似与频率成正比。
5. 若两个频点的模型不一致，显示“非理想/无法分类”，但仍显示原始R、X、|Z|和相位。

### 4.6 自动量程和频率

初始探测建议：

```text
频率 = 10 kHz
Rref = 1 kΩ
激励 = 200 mVpp
平均 = 1帧
```

根据初测结果调整：

1. 选择最接近 `|Zx|` 的Rref，使 `0.1 ≤ |Zx|/Rref ≤ 10`，最佳点接近1。
2. 对电容估值 `Cest`，选择：

```text
f_target ≈ 1 / (2*pi*Cest*Rref)
```

3. 对电感估值 `Lest`，选择：

```text
f_target ≈ Rref / (2*pi*Lest)
```

4. 频率从离散标定表中选择，建议首版：

```text
100 Hz、1 kHz、10 kHz、100 kHz、500 kHz
```

5. 若任一ADC通道接近饱和，降低激励；若幅值过低，增大激励或切换Rref。

### 4.7 校准

每个 `频率 × Rref档位` 保存一组复数校准参数：

- 参考电阻实际值；
- 两路通道复增益比；
- SHORT残余串联阻抗 `Zshort`；
- OPEN残余并联导纳 `Yopen`；
- 校准温度、时间和校准编号。

建议校准顺序：

1. SHORT：夹具短路，测量引线和开关的串联残余。
2. OPEN：夹具开路，测量输入和夹具的并联寄生。
3. LOAD：接入0.1%标准电阻，校准两路复增益比和量程比例。

初版修正模型：

```text
Z1 = Zraw - Zshort
Y1 = 1 / Z1 - Yopen
Zcorrected = 1 / Y1
```

LOAD用于求取每档复数比例系数。最终以标准R/C/L实测数据决定是否升级为完整三项误差模型。

### 4.8 通信协议

正式实现使用统一V2协议，不再以当前独立 `F9 26` 测电压协议作为最终接口。

主要命令：

| 命令 | 方向 | 用途 |
| --- | --- | --- |
| `MEASURE_START 0x40` | Black→Blue | 类型、频率、幅值、平均次数、校准编号 |
| `MEASURE_ACCEPTED 0x41` | Blue→Black | 实际频率、量程、预计完成时间 |
| `MEASURE_RESULT 0x42` | Blue→Black | V、I、相位、Z、R、X、L、C、Q |
| `MEASURE_ERROR 0x43` | Blue→Black | 错误阶段、ADC范围、超时和硬件故障 |
| `MEASURE_STATUS 0x47` | Blue→Black | 当前测量状态和进度 |

Black负责设置AD9910并等待稳定后再请求Blue采样。Blue负责选择Rref、同步采样和计算。协议结果必须增加或确认以下字段：

```text
range_id、reference_resistor_mohm、calibration_id、
vin_complex、current_complex、result_flags
```

迁移期间可以保留旧测量帧解析器，但新LCR页面只消费V2完整阻抗结果。

### 4.9 界面

LCR页面建议分为三块：

1. 设置：AUTO/R/L/C、AUTO/手动频率、激励、平均次数。
2. 主结果：元件类型和主要数值，例如 `L = 10.02 mH`。
3. 详细结果：频率、Rref、|Z|、相位、ESR、Q/D、校准编号和状态。

状态文本统一为：

```text
SETTING_SOURCE → SETTLING → RANGING → CAPTURING
→ PROCESSING → DONE / ERROR
```

## 5. 当前系统与目标系统差距

| 项目 | 当前实现 | 目标实现 |
| --- | --- | --- |
| 激励 | 固定1 MHz满幅正弦 | 自动频率和安全幅值 |
| ADC | PC0单通道 | 两路同步ADC |
| 算法 | 单路正弦幅值拟合 | 双路复数阻抗计算 |
| 输出 | RMS、Peak、Vpp | R、L、C、ESR、Q、相位 |
| 量程 | 无 | 多档Rref自动量程 |
| 校准 | 仅假定3.3 V参考 | OPEN/SHORT/LOAD复数校准 |
| 协议 | 独立ADC测量V1帧 | 统一V2 LCR协议 |
| 精度 | 未形成LCR指标 | 标定范围内≤1% |

## 6. 实施阶段

### 阶段一：双通道硬件和原始相量

- 确认第二ADC引脚和测量夹具接线。
- 完成两路同步ADC DMA。
- 屏幕显示两路幅值和相位差。
- 使用同一正弦信号分接两路，验证幅值比接近1、相位差接近0。

### 阶段二：单档阻抗测量

- 固定1 kΩ参考电阻和10 kHz激励。
- 实现复数Z、R、X、L、C计算。
- 使用100 Ω、1 kΩ、10 kΩ标准电阻验证公式和符号。

### 阶段三：自动分类和量程

- 增加Rref档位控制。
- 实现自动频率、激励幅值和双频确认。
- 增加AUTO/R/L/C界面。

### 阶段四：校准与1%验收

- 实现OPEN/SHORT/LOAD校准存储。
- 使用标准R、C、L覆盖目标范围。
- 建立误差表，收缩或扩展最终承诺量程。

### 阶段五：统一协议和产品化界面

- 切换到V2 `0x40～0x47`测量协议。
- 完成状态、错误、扫频和校准页面。
- 移除或隔离旧的单路ADC测试入口。

## 7. 验收标准

1. 同一信号接入两路ADC时，校准后幅值差≤0.2%，相位差≤0.2°。
2. 标准电阻测量误差≤1%，且电抗接近误差预算范围内的0。
3. 标准电容测量误差≤1%，X为负，两个频点计算出的C一致。
4. 标准电感测量误差≤1%，X为正，两个频点计算出的L一致。
5. 连续测量10次的重复性优于0.3%。
6. OPEN、SHORT、过量程和低信噪比均返回明确无效状态。
7. 更换量程或频率后不会沿用上一档校准参数。
8. 测量完成后信号源和ADC均恢复到可控状态。

## 8. 待确认事项

1. Blue开发板第二ADC引脚的实际可用性和外部引出位置。
2. 测量夹具准备采用继电器还是模拟开关切换Rref。
3. 首轮必须保证1%的R、C、L具体范围。
4. 实际测量激励经过哪一级放大器，以及夹具处可用的最大安全幅值。
5. 标准电阻、电容和电感的可用精度等级与数量。
